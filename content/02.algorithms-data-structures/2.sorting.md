# Сортировки

Здесь мы разберем все основные алгоритмы сортировки массивов.

## Bubble Sort

<b class="text-on-semi-light">Bubble Sort</b> — простой алгоритм сортировки, который многократно проходит по массиву, сравнивает соседние элементы и меняет их местами, если они в неправильном порядке.

**Основные характеристики:**

1. **Итеративный процесс:** каждый проход «всплывает» максимальный элемент в конец массива.
2. **Сложность:** `O(n²)` в худшем и среднем случаях, `O(n)` в лучшем (если массив уже отсортирован).
3. **Стабильность:** алгоритм стабильный (не меняет порядок равных элементов).

**Пример (TypeScript):**

```ts
function bubbleSort(arr: number[]): number[] {
  let n = arr.length;
  let swapped;
  do {
    swapped = false;
    for (let i = 1; i < n; i++) {
      if (arr[i - 1] > arr[i]) {
        [arr[i - 1], arr[i]] = [arr[i], arr[i - 1]];
        swapped = true;
      }
    }
    n--;
  } while (swapped);
  return arr;
}
```

**Применение:** обучающие примеры, очень маленькие массивы.

**Недостатки:** крайне неэффективен для больших данных.

## Insertion Sort

<b class="text-on-semi-light">Insertion Sort</b> — алгоритм, который строит отсортированный массив, вставляя элементы один за другим в правильное место.

**Основные характеристики:**

1. **Поэлементное построение:** каждый новый элемент вставляется в уже отсортированную часть.
2. **Сложность:** `O(n²)` в худшем и среднем случаях, `O(n)` в лучшем.
3. **Стабильность:** сохраняет порядок равных элементов.

**Пример (TypeScript):**

```ts
function insertionSort(arr: number[]): number[] {
  for (let i = 1; i < arr.length; i++) {
    let key = arr[i];
    let j = i - 1;
    while (j >= 0 && arr[j] > key) {
      arr[j + 1] = arr[j];
      j--;
    }
    arr[j + 1] = key;
  }
  return arr;
}
```

**Применение:** сортировка небольших массивов, часто используется в гибридных алгоритмах (например, в Timsort).

**Недостатки:** неэффективен для больших массивов.

## Selection Sort

<b class="text-on-semi-light">Selection Sort</b> — алгоритм сортировки, который на каждом шаге выбирает минимальный элемент и ставит его на нужное место.

**Основные характеристики:**

1. **Выбор минимума:** каждый проход ищет наименьший элемент и помещает его в начало.
2. **Сложность:** `O(n²)` всегда.
3. **Стабильность:** не является стабильным.

**Пример (TypeScript):**

```ts
function selectionSort(arr: number[]): number[] {
  for (let i = 0; i < arr.length; i++) {
    let minIndex = i;
    for (let j = i + 1; j < arr.length; j++) {
      if (arr[j] < arr[minIndex]) {
        minIndex = j;
      }
    }
    [arr[i], arr[minIndex]] = [arr[minIndex], arr[i]];
  }
  return arr;
}
```

**Применение:** простые учебные примеры.

**Недостатки:** медленный для больших массивов.

## Merge Sort

<b class="text-on-semi-light">Merge Sort</b> — алгоритм «разделяй и властвуй», который делит массив пополам, сортирует обе части и объединяет их.

**Основные характеристики:**

1. **Разделение:** массив рекурсивно делится на части.
2. **Слияние:** отсортированные подмассивы объединяются.
3. **Сложность:** всегда `O(n log n)`.
4. **Стабильность:** стабильный.

**Пример (TypeScript):**

```ts
function mergeSort(arr: number[]): number[] {
  if (arr.length <= 1) return arr;
  const mid = Math.floor(arr.length / 2);
  const left = mergeSort(arr.slice(0, mid));
  const right = mergeSort(arr.slice(mid));
  return merge(left, right);
}

function merge(left: number[], right: number[]): number[] {
  const result: number[] = [];
  let i = 0,
    j = 0;
  while (i < left.length && j < right.length) {
    if (left[i] <= right[j]) result.push(left[i++]);
    else result.push(right[j++]);
  }
  return [...result, ...left.slice(i), ...right.slice(j)];
}
```

**Применение:** большие массивы, внешняя сортировка.

**Недостатки:** требует дополнительной памяти `O(n)`.

## Quick Sort

<b class="text-on-semi-light">Quick Sort</b> — быстрый алгоритм сортировки, который выбирает «опорный элемент» (pivot) и распределяет элементы вокруг него.

**Основные характеристики:**

1. **Разделение массива:** элементы меньше pivot идут влево, больше — вправо.
2. **Рекурсия:** процесс повторяется для подмассивов.
3. **Сложность:** `O(n log n)` в среднем, `O(n²)` в худшем.
4. **Стабильность:** не стабильный.

**Пример (TypeScript):**

```ts
function quickSort(arr: number[]): number[] {
  if (arr.length <= 1) return arr;
  const pivot = arr[arr.length - 1];
  const left: number[] = [];
  const right: number[] = [];

  for (let i = 0; i < arr.length - 1; i++) {
    if (arr[i] < pivot) left.push(arr[i]);
    else right.push(arr[i]);
  }
  return [...quickSort(left), pivot, ...quickSort(right)];
}
```

**Применение:** стандартный выбор в практике.

**Недостатки:** плохо работает на уже отсортированных данных (если pivot выбирается неудачно).

## Heap Sort

<b class="text-on-semi-light">Heap Sort</b> — алгоритм, основанный на структуре данных «куча».

**Основные характеристики:**

1. **Использует max-heap:** на каждом шаге удаляется наибольший элемент.
2. **Сложность:** всегда `O(n log n)`.
3. **Стабильность:** не стабильный.

**Пример (TypeScript):**

```ts
function heapSort(arr: number[]): number[] {
  const n = arr.length;

  const heapify = (n: number, i: number) => {
    let largest = i;
    const left = 2 * i + 1;
    const right = 2 * i + 2;

    if (left < n && arr[left] > arr[largest]) largest = left;
    if (right < n && arr[right] > arr[largest]) largest = right;

    if (largest !== i) {
      [arr[i], arr[largest]] = [arr[largest], arr[i]];
      heapify(n, largest);
    }
  };

  for (let i = Math.floor(n / 2) - 1; i >= 0; i--) heapify(n, i);
  for (let i = n - 1; i > 0; i--) {
    [arr[0], arr[i]] = [arr[i], arr[0]];
    heapify(i, 0);
  }

  return arr;
}
```

**Применение:** когда важна предсказуемость времени работы.

**Недостатки:** медленнее на практике, чем QuickSort.

## Counting Sort

<b class="text-on-semi-light">Counting Sort</b> — алгоритм сортировки подсчётом, работающий с ограниченным диапазоном целых чисел.

**Основные характеристики:**

1. **Работает с целыми:** предполагается, что элементы принадлежат диапазону `0..k`.
2. **Сложность:** `O(n + k)`.
3. **Стабильность:** стабильный.

**Пример (TypeScript):**

```ts
function countingSort(arr: number[], maxVal: number): number[] {
  const count = new Array(maxVal + 1).fill(0);
  for (let num of arr) count[num]++;

  const result: number[] = [];
  for (let i = 0; i <= maxVal; i++) {
    while (count[i] > 0) {
      result.push(i);
      count[i]--;
    }
  }
  return result;
}
```

**Применение:** сортировка небольших диапазонов (например, оценки, возраст).

**Недостатки:** не подходит для больших диапазонов.

## Radix Sort

<b class="text-on-semi-light">Radix Sort</b> — алгоритм, который сортирует числа по разрядам (от младших к старшим).

**Основные характеристики:**

1. **Поразрядная сортировка:** на каждом шаге сортируется по одному разряду.
2. **Использует стабильную сортировку (обычно Counting Sort).**
3. **Сложность:** `O(d·(n + k))`, где `d` — количество разрядов.
4. **Стабильность:** стабильный.

**Пример (TypeScript):**

```ts
function radixSort(arr: number[]): number[] {
  const maxNum = Math.max(...arr);
  let exp = 1;

  while (Math.floor(maxNum / exp) > 0) {
    arr = countingSortByDigit(arr, exp);
    exp *= 10;
  }
  return arr;
}

function countingSortByDigit(arr: number[], exp: number): number[] {
  const output: number[] = new Array(arr.length).fill(0);
  const count = new Array(10).fill(0);

  for (let num of arr) {
    const index = Math.floor(num / exp) % 10;
    count[index]++;
  }

  for (let i = 1; i < 10; i++) count[i] += count[i - 1];

  for (let i = arr.length - 1; i >= 0; i--) {
    const index = Math.floor(arr[i] / exp) % 10;
    output[count[index] - 1] = arr[i];
    count[index]--;
  }

  return output;
}
```

**Применение:** сортировка длинных чисел, когда сравнение дорого.

**Недостатки:** требует стабильной сортировки на каждом шаге и неэффективен для строк/чисел с плавающей точкой.

## Bucket Sort

<b class="text-on-semi-light">Bucket Sort</b> — алгоритм сортировки, который распределяет элементы по «корзинам» (bucket), сортирует каждую корзину отдельно (часто с помощью встроенной сортировки или другой алгоритма), а затем объединяет результат.

**Основные характеристики:**

1. **Идеален для равномерно распределённых данных:** особенно для чисел с плавающей точкой на отрезке `[0, 1)`.
2. **Сложность:** в среднем `O(n + k)`, где `k` — число корзин; в худшем `O(n²)`.
3. **Стабильность:** зависит от используемой сортировки внутри корзин.

**Пример (TypeScript):**

```ts
function bucketSort(arr: number[], bucketSize: number = 5): number[] {
  if (arr.length === 0) return arr;

  let minValue = Math.min(...arr);
  let maxValue = Math.max(...arr);

  const bucketCount = Math.floor((maxValue - minValue) / bucketSize) + 1;
  const buckets: number[][] = Array.from({ length: bucketCount }, () => []);

  for (const num of arr) {
    const idx = Math.floor((num - minValue) / bucketSize);
    buckets[idx].push(num);
  }

  return ([] as number[]).concat(
    ...buckets.map((bucket) => bucket.sort((a, b) => a - b))
  );
}
```

**Применение:** сортировка чисел с плавающей точкой, равномерно распределённых данных.

**Недостатки:** неэффективен для данных с большой разницей в значениях или неравномерным распределением.

## Shell Sort

<b class="text-on-semi-light">Shell Sort</b> — оптимизация сортировки вставками, использующая «шагающий» разрыв для сравнения элементов.

**Основные характеристики:**

1. **Уменьшает разрыв:** сначала сортируются элементы на большом расстоянии, затем разрыв уменьшается.
2. **Сложность:** сильно зависит от выбранной последовательности шагов; худший случай — `O(n²)`, лучший — около `O(n log² n)`.
3. **Стабильность:** алгоритм нестабильный.

**Пример (TypeScript):**

```ts
function shellSort(arr: number[]): number[] {
  let n = arr.length;
  let gap = Math.floor(n / 2);

  while (gap > 0) {
    for (let i = gap; i < n; i++) {
      let temp = arr[i];
      let j = i;
      while (j >= gap && arr[j - gap] > temp) {
        arr[j] = arr[j - gap];
        j -= gap;
      }
      arr[j] = temp;
    }
    gap = Math.floor(gap / 2);
  }
  return arr;
}
```

**Применение:** когда требуется быстрый, но простой в реализации алгоритм.

**Недостатки:** хуже современных `O(n log n)` алгоритмов.

## TimSort

<b class="text-on-semi-light">TimSort</b> — гибридная сортировка (основана на Merge Sort и Insertion Sort), используемая в Python, Java и других языках.

**Основные характеристики:**

1. **Использует natural runs:** алгоритм ищет уже отсортированные подпоследовательности и эффективно объединяет их.
2. **Сложность:** `O(n log n)` в худшем случае, близко к `O(n)` на почти отсортированных данных.
3. **Стабильность:** алгоритм стабильный.

**Пример (TypeScript, упрощённая версия):**

```ts
function insertionSort(arr: number[], left: number, right: number): void {
  for (let i = left + 1; i <= right; i++) {
    let key = arr[i];
    let j = i - 1;
    while (j >= left && arr[j] > key) {
      arr[j + 1] = arr[j];
      j--;
    }
    arr[j + 1] = key;
  }
}

function merge(arr: number[], l: number, m: number, r: number): void {
  const left = arr.slice(l, m + 1);
  const right = arr.slice(m + 1, r + 1);
  let i = 0,
    j = 0,
    k = l;

  while (i < left.length && j < right.length) {
    arr[k++] = left[i] <= right[j] ? left[i++] : right[j++];
  }

  while (i < left.length) arr[k++] = left[i++];
  while (j < right.length) arr[k++] = right[j++];
}

function timSort(arr: number[]): number[] {
  const n = arr.length;
  const RUN = 32;

  for (let i = 0; i < n; i += RUN) {
    insertionSort(arr, i, Math.min(i + RUN - 1, n - 1));
  }

  for (let size = RUN; size < n; size *= 2) {
    for (let left = 0; left < n; left += 2 * size) {
      const mid = left + size - 1;
      const right = Math.min(left + 2 * size - 1, n - 1);
      if (mid < right) merge(arr, left, mid, right);
    }
  }
  return arr;
}
```

**Применение:** используется в стандартных библиотечных сортировках (Python `sort()`, Java `Arrays.sort()` для объектов).

**Недостатки:** сложен в реализации по сравнению с другими сортировками.
