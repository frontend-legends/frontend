## Быстрый обзор

В этом разделе мы рассмотрим ключевые алгоритмы и структуры данных, которые чаще всего встречаются на собеседованиях и в практических задачах.

- **Сортировки:** Bubble Sort, Insertion Sort, Selection Sort, Merge Sort, Quick Sort, Heap Sort, Counting Sort, Radix Sort, Bucket Sort, Shell Sort, TimSort.
- **Поиск:** Линейный поиск, Бинарный поиск, Интерполяционный поиск.
- **Структуры данных:** Стек, Очередь, Дек, Связный список, Хеш-таблицы.
- **Деревья:** Binary Tree, Binary Search Tree (BST), AVL Tree, Red-Black Tree, Segment Tree, Fenwick Tree (BIT), Trie, Heap.
- **Графы:** BFS, DFS, Топологическая сортировка, Кратчайшие пути (Dijkstra, Bellman-Ford, Floyd-Warshall, A\*), Минимальное остовное дерево (Kruskal, Prim).
- **Жадные алгоритмы:** Interval Scheduling, Huffman Coding.
- **Динамическое программирование:** Fibonacci, Knapsack, Longest Common Subsequence, Longest Increasing Subsequence, Matrix Chain Multiplication.
- **Алгоритмы на строках:** KMP, Rabin-Karp, Z-алгоритм, Suffix Array/Tree, Trie-based поиск.
- **Алгоритмы на массивах:** Two Pointers, Sliding Window, Prefix Sum, Kadane’s Algorithm.
- **Хэширование и кэш:** HashMap/Set, LRU Cache, LFU Cache.
- **Дополнительно:** Union-Find (DSU), Backtracking (N-Queens, Sudoku), Bit Manipulation.
