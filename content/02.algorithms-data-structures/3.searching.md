# Поиск

## Линейный поиск (Linear Search)

<b class="text-on-semi-light">Linear Search</b> — простой алгоритм поиска, который проверяет каждый элемент по очереди.

**Основные характеристики:**

1. **Прямой перебор:** проходит массив от начала до конца.
2. **Сложность:** `O(n)`.
3. **Подходит для:** неотсортированных массивов.

**Пример (TypeScript):**

```ts
function linearSearch(arr: number[], target: number): number {
  for (let i = 0; i < arr.length; i++) {
    if (arr[i] === target) return i;
  }
  return -1;
}
```

**Применение:** когда данные не отсортированы или массив небольшой.

**Недостатки:** медленный для больших массивов.

## Бинарный поиск (Binary Search)

<b class="text-on-semi-light">Binary Search</b> — эффективный алгоритм для отсортированных массивов.

**Основные характеристики:**

1. **Деление пополам:** на каждом шаге отбрасывает половину массива.
2. **Сложность:** `O(log n)`.
3. **Требование:** массив должен быть отсортирован.

**Пример (TypeScript):**

```ts
function binarySearch(arr: number[], target: number): number {
  let left = 0;
  let right = arr.length - 1;

  while (left <= right) {
    const mid = Math.floor((left + right) / 2);

    if (arr[mid] === target) return mid;
    if (arr[mid] < target) left = mid + 1;
    else right = mid - 1;
  }

  return -1;
}
```

**Применение:** быстрый поиск в отсортированных массивах.

**Недостатки:** работает только с отсортированными данными.

## Интерполяционный поиск (Interpolation Search)

<b class="text-on-semi-light">Interpolation Search</b> — улучшенный вариант бинарного поиска для равномерно распределённых данных.

**Основные характеристики:**

1. **Формула позиции:** индекс выбирается пропорционально значению (`pos = left + (target - arr[left]) * (right - left) / (arr[right] - arr[left])`).
2. **Сложность:** `O(log log n)` в лучшем случае, `O(n)` в худшем.
3. **Подходит для:** равномерно распределённых отсортированных массивов.

**Пример (TypeScript):**

```ts
function interpolationSearch(arr: number[], target: number): number {
  let left = 0;
  let right = arr.length - 1;

  while (left <= right && target >= arr[left] && target <= arr[right]) {
    if (left === right) {
      return arr[left] === target ? left : -1;
    }

    const pos =
      left +
      Math.floor(
        ((target - arr[left]) * (right - left)) / (arr[right] - arr[left])
      );

    if (arr[pos] === target) return pos;
    if (arr[pos] < target) left = pos + 1;
    else right = pos - 1;
  }

  return -1;
}
```

**Применение:** базы данных, индексы, когда значения распределены равномерно.

**Недостатки:** неэффективен для неравномерно распределённых данных.

## Экспоненциальный поиск (Exponential Search)

<b class="text-on-semi-light">Exponential Search</b> — сочетает экспоненциальное расширение диапазона и бинарный поиск.

**Основные характеристики:**

1. **Удвоение диапазона:** находит диапазон, где может находиться элемент.
2. **Затем бинарный поиск:** выполняется внутри найденного диапазона.
3. **Сложность:** `O(log i)`, где `i` — позиция элемента.

**Пример (TypeScript):**

```ts
function exponentialSearch(arr: number[], target: number): number {
  if (arr[0] === target) return 0;

  let i = 1;
  while (i < arr.length && arr[i] <= target) i *= 2;

  return binarySearchInRange(arr, target, i / 2, Math.min(i, arr.length - 1));
}

function binarySearchInRange(
  arr: number[],
  target: number,
  left: number,
  right: number
): number {
  while (left <= right) {
    const mid = Math.floor((left + right) / 2);
    if (arr[mid] === target) return mid;
    if (arr[mid] < target) left = mid + 1;
    else right = mid - 1;
  }
  return -1;
}
```

**Применение:** эффективен для бесконечных или очень больших массивов.

**Недостатки:** требует отсортированных данных.

## Джамп-поиск (Jump Search)

<b class="text-on-semi-light">Jump Search</b> — алгоритм, который прыгает блоками фиксированного размера.

**Основные характеристики:**

1. **Шаг:** обычно `√n`.
2. **Сначала прыжки:** ищем блок, где может быть элемент.
3. **Потом линейный поиск:** внутри блока.

**Сложность:** `O(√n)`.

**Пример (TypeScript):**

```ts
function jumpSearch(arr: number[], target: number): number {
  const n = arr.length;
  let step = Math.floor(Math.sqrt(n));
  let prev = 0;

  while (arr[Math.min(step, n) - 1] < target) {
    prev = step;
    step += Math.floor(Math.sqrt(n));
    if (prev >= n) return -1;
  }

  while (arr[prev] < target) {
    prev++;
    if (prev === Math.min(step, n)) return -1;
  }

  return arr[prev] === target ? prev : -1;
}
```

**Применение:** эффективен для отсортированных массивов среднего размера.

**Недостатки:** хуже бинарного поиска по асимптотике.
