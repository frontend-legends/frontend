# CSS

**CSS (Cascading Style Sheets)** — это не «раскрашивание div'ов». На senior-уровне CSS — это понимание **алгоритма каскада**, layout-моделей (BFC, stacking context), архитектуры стилей (@layer, CSS Modules, BEM) и оптимизации рендеринга. Глубокое знание CSS отделяет инженера, который **проектирует** интерфейсы, от того, кто подбирает стили методом проб и ошибок.

---

## Зачем это нужно на собесе

На mid+ и senior собеседованиях CSS — это не «сверстай макет», а проверка понимания платформы:

- объяснить, **как работает каскад** и почему стили не применяются (specificity, origin, order);
- знать, что такое **BFC** и когда margin collapse не происходит;
- понимать, почему **z-index: 9999 не работает** (stacking context);
- уверенно выбирать **Flexbox vs Grid** и объяснить edge cases (`flex: 1`, `auto-fill` vs `auto-fit`);
- разбираться в **container queries**, `@layer` и нативном CSS nesting;
- уметь объяснить, почему анимировать **только `transform` и `opacity`** (composite-only);
- обосновать выбор **CSS-архитектуры**: BEM, CSS Modules, utility-first, CSS-in-JS.

---

## Что даёт глубокое знание CSS

- Пишешь **предсказуемые стили** без `!important` и специфичности-войн.
- Строишь **адаптивные layout'ы**, которые работают на всех viewport'ах без хаков.
- Создаёшь **производительные анимации**, которые не дёргают UI.
- Выбираешь **правильную архитектуру** стилей для масштаба проекта.
- Понимаешь связь между CSS и **производительностью рендеринга**: CRP, reflow, repaint, composite.

> CRP (Critical Rendering Path) и CSSOM подробно разобраны в [Структура и рендеринг](/content/html/structure).

## Что мы разберём

В этом разделе — 9 тем, от основ каскада до архитектуры стилей:

1. **Каскад и стилизация** — cascade algorithm, специфичность, наследование, box model, margin collapse
2. **Селекторы и at-rules** — псевдоклассы, `:has()`, @layer, @property, CSS-функции
3. **Раскладка и поток** — BFC, позиционирование, stacking context, Flexbox, Grid
4. **Адаптивный дизайн** — media queries, container queries, viewport units, fluid typography
5. **Современный CSS** — custom properties, nesting, `:has()`, oklch, performance
6. **Анимации и переходы** — transitions, @keyframes, производительность, scroll-driven
7. **Типографика и шрифты** — @font-face, font-display, variable fonts, text overflow
8. **Визуальные эффекты** — градиенты, тени, filter, clip-path, blend modes
9. **CSS-архитектура** — BEM, CSS Modules, CSS-in-JS, utility-first, design tokens
